Tabs

Use tabs to group your properties in way that improves the editor experience.

No tabs means a lot of scrolling

Not clear to editors where something can be found.

Tabs

  • Group them by functionality
  • Use groups to combine them in side a tab
  • Order tabs in so frequently edited fields come first
  • Observe your editors to optimize structure.

Group by function => all meta data props in one tab

Group inside tab => seo and open graph

Order : your main content fields should be the first tab.
